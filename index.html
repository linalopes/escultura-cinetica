Aqui um texto blablabla de como usar as ferramentas

1. Pontas dos arames

Torça as ponta dos arames de modo a fazer um pequeno anel. Fique atento para não criar espaço para que as peças se soltem depois. Se necessário faça mais de uma volta.

2. Conectores

Faça conectores com pedaços pequenos do arame. No final, os conectores vão ficar parecendo um S
